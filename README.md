# Senderkataster-Modder
Die Erweiterung hat die folgenden Funktionen:

- Fügt Netzbetreiber-Informationen basierend auf den Angaben Protokoll(e) und Sendeleistung hinzu. Die Informationen können stimmen, müssen sie aber nicht immer. 
- Erlaubt es, weiter in die Karte hinein zu zoomen (Level 22 statt 15). In der Standardkonfiguration konnte es vorkommen, dass Marker zu nahe beisammen waren, um jeden anzuklicken.
- Deaktiviert den automatischen Zoom, wenn auf einen Marker geklickt wird.
- Versteckt das Informationsfeld unter dem Adressfeld automatisch.
- Erhöht die Informationsdichte der Pop-ups. Je nach Display können bis zu drei Station-Einträge auf einmal dargestellt werden.
## Installation der Entwicklungsversion
### Chromium
- Lade das Repo herunter und entpacke es.
- Besuche die Seite `chrome://extensions/`, aktiviere den Entwicklermodus oben rechts.
- Es erscheint oben links der Button „Entpackte Erweiterung laden“. Klicke auf ihn.
- Wähle den entpackten Chrome-Ordner aus.
- Die Erweiterung steht dir bei künftigen Besuchen auf der Senderkataster-Webseite zur Verfügung.
### Firefox
- Lade das Repo herunter und entpacke es.
- Besuche die Seite `about:debugging#/runtime/this-firefox`.
- Klicke auf den Button „Temporäres Add-on" laden.
- Wähle das ZIP-Archiv im Firefox-Ordner aus.
- Die Erweiterung steht dir zumindest temporär bei künftigen Besuchen auf der Senderkataster-Webseite zur Verfügung.chen auf der Senderkataster Webseite zur Verfügung.
